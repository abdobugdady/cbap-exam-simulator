# 🎓 CBAP Exam Simulator — BABOK v3

A fully self-contained, browser-based exam preparation tool for the **Certified Business Analysis Professional (CBAP)** exam.

## ✨ Features

| Feature | Details |
|---|---|
| 📝 Full Exam Mode | 150 questions · 3.5-hour timer · CBAP format |
| ⚡ Quick Quiz | 20 questions · instant feedback · streak tracking |
| 🃏 Flashcards | 80 key BABOK v3 concepts with flip animation |
| 📊 Analytics | Per-knowledge-area progress tracking |
| 💾 Persistence | Progress saved locally in your browser |

- **150 questions** (120 scored + 30 unscored)
- **3.5 hours** time limit
- **Passing score:** ~75% (IIBA does not publish exact passing score)
- Based on **BABOK v3** (current standard)
